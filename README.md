Description
===========

tualo-webmail is a web IMAP client.



Requirements
============

* [node.js](http://nodejs.org/) -- v0.8.0 or newer
* An IMAP Account


Installation
============

    npm install tualo-webmail
    or 
    npm install https://github.com/tualo/tualo-webmail.git

How to run
============

    Start the server with the commandline:
    node app.js
